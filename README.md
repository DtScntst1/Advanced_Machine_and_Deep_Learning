This repository contains my advanced projects focusing on Deep Learning, Artificial Neural Networks (ANNs), Convolutional Neural Networks (CNNs), and complex ensemble methods. 

## Overview
- **Objective:** To tackle complex, non-linear problems such as image classification, natural language processing (NLP), and advanced predictive modeling using multi-layered neural architectures.
- **Approach:** Designing robust deep neural network architectures, applying regularization techniques (Dropout, Batch Normalization) to prevent overfitting, and optimizing learning rates.

## Tech Stack
- **Deep Learning Frameworks:** TensorFlow, Keras
- **Advanced ML:** XGBoost, LightGBM
- **Environment:** GPU-accelerated Jupyter Notebooks
